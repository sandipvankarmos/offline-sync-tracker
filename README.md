# myWork.AI - Offline Task Tracker

A modern, offline-first task tracker built with vanilla JavaScript and Tailwind CSS.

## Features

- **CRUD Operations** - Create, edit, and delete tasks with intuitive modals
- **Offline-First** - All data persisted to localStorage, works without internet
- **Priority Levels** - High, Medium, Low with color-coded badges
- **Search & Filter** - Real-time search, filter by status/priority/phase
- **Dark Mode** - Toggle with system preference detection
- **Keyboard Shortcuts** - `N` (new task), `Ctrl+K` (search), `Esc` (close)
- **Toast Notifications** - Feedback for all user actions
- **Responsive Design** - Works on desktop and mobile

## Live Demo

Visit: [amazing-queijadas-f76de0.netlify.app](https://amazing-queijadas-f76de0.netlify.app)

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Chart.js for analytics
- LocalStorage for persistence

## Getting Started

Simply open `index.html` in your browser, or deploy to any static hosting service.
